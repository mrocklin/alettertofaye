This is a brief letter about basic internet security. It is for someone who does not know very much about computers.

My father received this e-mail from one of his bridge partners

Faye Wrote:

Having experienced 2 virus attacks within 2 months, I am less than thrilled with Mcafee anti virus software. The next time you speak with Matt (if you remember) please ask him if he has a recommendation for a better anti virus program Thanks, Faye

Faye's frustrations are both very common and easy to prevent. I get an e-mail like this about once a month. After writing so many I've decided just to post my response online.

Hi Faye,

I would do the following:

I would update your internet browser to a newer edition. An internet browser is a program like "Internet Explorer" or "Safari"; it is the program you click on to access the intenet. Newer versions can detect most bad programs before they reach your computer. I highly recommend Google Chrome. You would want to make sure that you double click on the Chrome icon whenever you access the Internet rather than your old program.

If you're using Windows, you should check that it has been updating itself. If a little yellow window has been popping up in the lower right of your screen telling you to install updates then you should click on it and follow the instructions. This should be easy.

Most computer issues these days aren't the old sort of virus that McAffe was originally designed to handle. Things have changed since McAffe's day. If you do want an anti-virus program I think the standard solution these days is AVG. The free version is sufficient. If it were me I would also uninstall McAffe. While it does offer protection it also has the bad habit of slowing down the computer while it scans for viruses which, by default, it does every day. AVG would likely do this as well. If you hunt around in the program options you'll be able to find some setting to make the scans weekly or monthly.

Many problems arise from malicious programs that aren't viruses but are instead called "malware" or "spyware". These are usually but not always taken care of by anti-virus programs like AVG. You can also download and install more specialized programs to handle these issues if you think AVG isn't catching everything. A popular one today is MalawareBytes.

If your computer is ok now (doesn't have a virus) then I would install and use Google Chrome and forget about the issue. This is what I do. I don't even have an anti-virus program running and I've been fine for years. If something does happen then you should install and run AVG. If that doesn't work then you should install and run Malawarebytes.

Additionally, its good just to be mindful of installing any small program that a suspicious website offers to you. Before a bad program enters your computer it needs to get your permission. Most of the time this will be a little pop up window on a website that asks you to install something.

Of course, good websites need to do this too. For example a website that plays movies like Netflix will need you to install something. You should let it. Whenever a small pop up window comes up though you should become suspicious and make sure that you're on a good, reputable website. Chrome is pretty good about warning you when you're on a suspicious website but some suspicion on your part is good too.
